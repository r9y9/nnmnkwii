nnmnkwii.io.hts.load_question_set¶

nnmnkwii.io.hts.load_question_set(qs_file_name)[source]¶

Load HTS-style question and convert it to binary/continuous feature extraction regexes.

This code was taken from Merlin.

Parameters:qs_file_name (str) – Input HTS-style question file path
Returns:Binary/continuous feature extraction regexes.
Return type:(binary_dict, continuous_dict)

Examples

>>> from nnmnkwii.io import hts
>>> from nnmnkwii.util import example_question_file
>>> binary_dict, continuous_dict = hts.load_question_set(example_question_file())
